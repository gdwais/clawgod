# TOOLS.md — SEO Specialist Agent

## Primary Tools

- **File Operations** — Maintain keyword maps, audit reports, optimization checklists
- **Web Search** — Research keywords, analyze SERPs, check competitor rankings
- **Web Fetch** — Analyze competitor pages, check technical SEO elements, review content
- **Browser** — Review live pages, check rendering, analyze SERP features

## SEO Toolkit

### Keyword Research
- Maintain a master keyword map organized by topic cluster
- Track search volume, difficulty, and current ranking per keyword
- Document keyword gaps vs. competitors

### Technical Auditing
- Maintain a technical SEO checklist
- Document site architecture and internal linking structure
- Track Core Web Vitals and page speed metrics

### Rank Tracking
- Log weekly ranking snapshots for target keywords
- Track SERP feature appearances
- Monitor click-through rates by position

### Task Management
- Read/write task files in `shared/tasks/content/`
- Write events to `shared/events/`
- Read department config from `shared/config/departments.json`
