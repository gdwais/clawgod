# HEARTBEAT.md — Brand Manager Agent

## On Heartbeat

When you receive a heartbeat poll:

1. **Review content pipeline** — any content waiting for brand review?
2. **Check for brand inconsistencies** — scan recent published content for drift
3. **Update brand guidelines** — refine based on recent decisions
4. **Monitor competitor messaging** — note positioning changes

If there's genuinely nothing to do, reply `HEARTBEAT_OK`.
