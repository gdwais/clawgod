# WORKFLOW.md — PR Engine

## Pipeline Stages

```
IDENTIFIED → RESEARCHED → PITCHED → RESPONDED → BOOKED → COMPLETED
```

All opportunities (podcasts, conferences, awards, speaking, journalist pitches) follow this pipeline.

## Daily HARO/Connectively Scan

```
Check Connectively + Qwoted for new queries
    ↓
Filter: fintech, wealth management, financial services
    ↓
Draft response (under 200 words, lead with data/insight)
    ↓
Save to shared/pr/haro-responses/ for review
    ↓
Human approves → submit before deadline
```

## Podcast Outreach Flow

```
Research shows (Listen Notes + web_search)
    ↓
Qualify: audience match, accepts guests, active
    ↓
Find host contact info
    ↓
Draft personalized pitch (reference specific episode)
    ↓
Save to shared/pr/drafts/ for review
    ↓
Human approves → send
    ↓
Log to shared/pr/tracking/pitches.json
    ↓
Follow up once (7 days later)
    ↓
If response → move to RESPONDED / BOOKED
If no response → archive, move on
```
